📄MMA 실시간 점수 시스템 - 실행 가이드 (의뢰인용)

📌 시스템 개요
이 웹 애플리케이션은 본부 1명 + 심판 3명이 실시간으로 점수를 송수신할 수 있는 시스템입니다.
- 서버 실행 기기: 노트북 (이 폴더가 위치한 기기)
- 사용자 기기: 본부/심판용 태블릿 또는 핸드폰
- 조건: 모든 기기는 같은 Wi-Fi에 연결되어야 합니다.

🧰 실행 전 준비 사항
1. 자바 설치 (필수)
- Java 17 이상이 설치되어 있어야 합니다. 없다면 Java 17 다운로드 링크에서 설치하세요.
 -> 설치 여부 확인: java -version

2. MySQL 설치 (필수)
- MySQL 8.x 설치 (포트는 3306 권장)
- 아래와 같이 사용자 계정 및 DB를 생성해주세요:
항목	값 (예시)
데이터베이스 이름	mma_scoring
사용자명	mma_user
비밀번호	mma_pass123

├── backend-0.0.1-SNAPSHOT.jar      ← 백엔드 실행 파일

    │   ├── index-*.js
    │   └── ...
